## Supported Ciphers
* Caesar
* Vigenere
* Atbash
* Vernam

## Quick start
1. Install dependencies:

    ```shell
    pip install -r requirements.txt
    ```

2. Run the GUI:
      ```shell
    python MainGUI.py
    ```
